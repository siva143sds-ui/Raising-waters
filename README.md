## Rising Waters: A Machine Learning Approach to Flood Predic
## Project Description:

Rising Waters is a Machine Learning-based flood prediction system that analyzes weather and rainfall data to predict the possibility of flood risks. The project uses different classification algorithms to identify patterns in environmental data and provide accurate flood predictions.

The system uses important weather parameters such as rainfall, cloud visibility, and meteorological conditions as input. Machine Learning models like Decision Tree, Random Forest, K-Nearest Neighbors (KNN), and XGBoost are trained and compared to find the best-performing model.

The selected model is integrated with a Flask web application where users can enter weather details and get instant flood risk predictions. This project aims to support early warning systems and improve disaster preparedness using Artificial Intelligence and Data Science techniques.


## Objectives

- To predict flood risks using Machine Learning techniques.
- To analyze weather and rainfall patterns affecting floods.
- To compare different ML algorithms and select the best model.
- To develop a simple web application for flood prediction.
